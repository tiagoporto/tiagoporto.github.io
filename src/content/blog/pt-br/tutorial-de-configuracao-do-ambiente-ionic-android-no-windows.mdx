Esse post vai em resposta à um pedido feito por um desenvolvedor em adição a minha palestra
[Aplicativos híbridos com Ionic. Você também pode começar a desenvolver agora!](https://tiagoporto.com/pt-br/blog/aplicativos-hibridos-com-ionic-voce-tambem-pode-comecar-a-desenvolver-agora-talk).

Então, vamos direto ao que interessa.

1.  Precisamos do <span lang="en-US">Ionic</span> e <span lang="en-US">Apache Cordova</span>,
    se ainda não os instalou, vamos lá.

    - Ambos são pacotes do <span lang="en-US">Node</span> e para isso precisamos instalá-lo.
      Faça o download em https://nodejs.org/en/download.
    - **Não desmarque <abbr title="Node Package Manager" lang="en-US">NPM</abbr>**.

      ![Tela de setup do Nodejs](/blog/ionic-android-enviroment-tutorial/node-setup-screen.png)

      _Não sabe o que é <span lang="en-US">Node</span> e <abbr title="Node Package Manager" lang="en-US">NPM</abbr>.
      Leia: [O que é a NPM do Node.js](http://nodebr.com/o-que-e-a-npm-do-nodejs/)._

2.  Com o <span lang="en-US">Node</span> instalado, agora vamos abrir o terminal e
    instalar o <span lang="en-US">Ionic</span> e o <span lang="en-US">Cordova</span>.

    - Execute o comando:

      ```bash
      npm install -g cordova ionic
      ```

      ![Terminal com instalação do Ionic sobre página de Getting Started do Ionic em background](/blog/ionic-android-enviroment-tutorial/ionic-install.png)

3.  Iniciaremos a configuração do ambiente para <span lang="en-US">Android</span>.
    Vamos instalar o <abbr title="Java Development Kit">JDK</abbr> do Java.

        - Acesse [Java Downloads](http://www.oracle.com/technetwork/java/javase/downloads/index.html),
          selecione <abbr title="Java Development Kit">JDK</abbr> download, faça o download
          da versão compatível com seu <abbr title="Sistema Operacional">SO</abbr> (32 ou 64 bits).
          _Veja o caminho de onde você instalou, será necessário no próximo passo._
        - Agora vamos criar a variável `JAVA_HOME` no ambiente do <span lang="en-US">Windows</span>,
          abra a tela de Sistema com as teclas <kbd lang="en-US">windows</kbd> + <kbd lang="en-US">pause/break</kbd>
          ou vá em `Painel de Controle / Sistema e Segurança / Sistema`.
        - Clique em `Configurações avançadas do sistema`.
        - Selecione `Variáveis de ambiente`.
        - Selecione `Nova`.
        - Na nova janela, em nome da variável coloque `JAVA_HOME`, e no valor coloque
          o caminho onde você instalou o <abbr title="Java Development Kit">JDK</abbr>.
          Por exemplo, `C:\Program Files\Java\jdk1.8.0_65`.
        - Salve.

          ![Janelas com o caminho para editar variáveis de ambiente no Windows](/blog/ionic-android-enviroment-tutorial/adding-system-var.png)

        - Selecione a variável <code lang="en-US">Path</code>, e clique em editar,
          no campo valores da variável, adicione `;%JAVA_HOME%\bin` ao final do valor.
          Não se esqueça do `;`.
        - Salve.

          ![Tela de editar variável do Windows](/blog/ionic-android-enviroment-tutorial/adding-bin-system-var.png)

4.  Com o Java configurado, agora precisamos do <span lang="en-US">Apache Ant</span>,
    o responsável pelo <em lang="en-US">build</em>. Faça o download
    em <a href="http://ant.apache.org/bindownload.cgi" lang="en-US">Ant Download</a>.

    - Após concluído o download, extraia o diretório para a raiz de `C:`.
    - Mais uma vez precisamos atualizar a variável <code lang="en-US">Path</code>
      acrescentando o <span lang="en-US">Ant</span>.
      Abra as variáveis de ambiente do <span lang="en-US">Windows</span>.
    - Selecione `Variáveis de ambiente`.
    - Selecione a variável <code lang="en-US">Path</code> e clique em editar.
    - Acrescente o caminho até a pasta bin do <span lang="en-US">Ant</span>,
      por exemplo `;C:\apache-ant-1.9.6\bin`.
      E não se esqueça do `;`.
    - Salve

      ![Tela de editar variável do Windows acrescentando o caminho do Ant](/blog/ionic-android-enviroment-tutorial/adding-ant-system-var.png)

5.  O próximo passo é a instalação do <abbr title="Software Development Kit">SDK</abbr>
    do <span lang="en-US">Android</span>, a <abbr title="Application Programming Interface">API</abbr>
    que fornece as bibliotecas e ferramentas necessárias para <em lang="en-US">build</em>,
    teste e <em lang="en-US">debug</em> para <span lang="en-US">Android</span>.

    - Faça o download em <a href="http://developer.android.com/sdk/index.html#Other" lang="en-US">Android SDK</a>,
      selecione o **.exe (Recommended)**.
    - Após instalado, precisamos criar a variável `ANDROID_HOME`, necessária para
      o Cordova. Abra as variáveis de ambiente.
    - Selecione `Variáveis de ambiente`.
    - Clique em Nova.
    - Na nova janela em nome da variável coloque `ANDROID_HOME`, no valor da
      variável coloque o caminho até a pasta `android-sdk` que você acabou de instalar,
      por exempo: `C:\Android\android-sdk`.
    - Salve.

      ![Janela do Windows de Nova Variável do Sistema](/blog/ionic-android-enviroment-tutorial/android-home-var.png)

    - Voltando às variáveis de ambiente, selecione a variável <code lang="en-US">Path</code>,
      e clique em editar.
    - Acrescente `;%ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools` ao valor da variável.
    - Salve.
    - Estamos quase lá, agora precisamos instalar os pacotes do <span lang="en-US">Android</span>.
      Abra o arquivo <code lang="en-US">SDK Manager.exe</code>, localizado dentro da pasta `android-sdk`.
    - Por padrão o <strong lang="en-US">Tools/Android SDK Tools</strong> já está instalado.
      Se por algum motivo não estiver, selecione-o.
    - Selecione <strong lang="en-US">Tools/Android SDK Platform-tool</strong>, <strong lang="en-US">Tools/SDK build-tools</strong>
      (uma versão acima da 22)
    - Na **API** selecione a mesma versão que você selecionou no <strong lang="en-US">Build Tools</strong>.
    - Em Extras, marque <strong lang="en-US">Google USB Driver</strong>,
      necessário para debugar direto no dispositivo.
    - Clique em instalar.

      ![Painel do Android SDK Manager](/blog/ionic-android-enviroment-tutorial/android-sdk-manager-panel.png)

    - Ao final das instalações nosso ambiente está configurado.

6.  Vamos agora adicionar a plataforma <span lang="en-US">Android</span> ao aplicativo.

    - Abra o diretório do aplicativo pelo terminal.
    - Execute o commando:

      ```bash
      ionic platform add android
      ```

      ![Terminal instalando a plataforma Android no projeto Ionic](/blog/ionic-android-enviroment-tutorial/ionic-platform-add-android.png)

7.  Vamos testar o <em lang="en-US">build</em> do `.apk`.

    - Execute:

      ```bash
      ionic build android
      ```

    - Após finalizado o processo, o caminho do `.apk` gerado é exibido no terminal.

      ![Terminal rodando build do Android](/blog/ionic-android-enviroment-tutorial/ionic-build-android.png)

8.  Podemos também debugar o aplicativo direto no dispositivo. Conecte o dispositivo
    via cabo <abbr title="Universal Serial Bus">USB</abbr>, certifique de ter
    habilitado o <strong lang="en-US">USB debugging</strong> no dispositivo.

    - Execute:

      ```bash
      ionic run android
      ```

      ![Dispositivo Android rodando o app, e no fundo, monitor com painel do Developer tools e um terminal](/blog/ionic-android-enviroment-tutorial/run-android.jpg)

É isso aí, bom desenvolvimento!
