Em vista à demanda por novos <abbr title="Cadastro Nacional da Pessoa Jurídica">CNPJ</abbr>s
e a possibilidade do esgotamento dos números disponíveis, em 15 de outubro de 2024
a Receita Federal Brasileira publicou uma instrução normativa
([Instrução Normativa RFB nº 2.229](https://normasinternet2.receita.fazenda.gov.br/#/consulta/externa/141102))
alterando o formato do <abbr title="Cadastro Nacional da Pessoa Jurídica">CNPJ</abbr>.
O novo formato agora incluirá letras e números.

O novo número continuará a ter 14 posições. As doze primeiras compostas por
letras e números e as duas últimas posições correspondentes aos dígitos verificadores
continuarão a ser numéricas.

## Compatibilidade

O novo formato é compatível com o anterior e todos os <abbr title="Cadastro Nacional da Pessoa Jurídica">CNPJ</abbr>s
numéricos existentes permanecerão válidos.

## Quando entra em vigor?

Pela atual Instrução Normativa a previsão é julho de 2026, se não houver adiamentos.

## Algoritmo

## Ajustes técnicos

## Banco de dados

Os campos de <abbr title="Cadastro Nacional da Pessoa Jurídica">CNPJ</abbr> precisam ser ajustados
para alfanuméricos caso sejam numéricos.

### Interface

#### Máscara de validação

Como mencionado, a máscara do novo <abbr title="Cadastro Nacional da Pessoa Jurídica">CNPJ</abbr> foi modificada, será necessário atualizar
a máscara de validação dos <span lang="en">`<input>`</span>.

O novo <span lang="en">pattern</span>:

`[A-Z0-9]{2}.[A-Z0-9]{3}.[A-Z0-9]{3}/[A-Z0-9]{4}-[0-9]{2}`

#### Algoritmo de validação

O algoritmo de validação precisa ser atualizado para validar um <abbr title="Cadastro Nacional da Pessoa Jurídica">CNPJ</abbr>
alfanumérico.

Para facilitar, disponibilizei uma <span lang="en">lib</span> em <span lang="en">TypeScript</span> com o novo algoritmo
([gerador-validador-cnpj](https://github.com/tiagoporto/gerador-validador-cpf/tree/main/packages/gerador-validador-cnpj)).

![](/blog/cnpj/gerador-validador-cnpj.png)

Basta instalar com <abbr title="Node Package Manager" lang="en">NPM</abbr>:

```bash
npm install gerador-validador-cnpj -D
```

E usar:

```js
import { validate as validadeCNPJ } from 'gerador-validador-cnpj'

const isCNPJValid = validadeCNPJ('01234567890000')
const isCNPJAlphanumericValid = validadeCNPJ('9ZW2JIM2OWTG85', {
  validateAlphanumeric: true
})
const isFormattedCNPJValid = validadeCNPJ('01.234.567/8900-00')
const isFormattedCNPJAlphanumericValid = validadeCNPJ('8I.S4O.LPO/PRD7-81', {
  validateAlphanumeric: true
})
```

Se estiver usando <span lang="en">Deno</span>, não precisa instalar, só importar direto:

```ts
import { validate } from 'jsr:@tiagoporto/gerador-validador-cnpj'
```

Documentação completa no [<abbr title="Node Package Manager" lang="en">NPM</abbr>](https://www.npmjs.com/package/gerador-validador-cnpj)
ou [<abbr title="JavaScript registry" lang="en">JSR</abbr>](https://jsr.io/@tiagoporto/gerador-validador-cnpj).
