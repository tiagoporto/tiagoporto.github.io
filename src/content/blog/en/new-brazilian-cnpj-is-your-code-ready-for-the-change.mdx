Given the demand for new <abbr title="National Registry of Legal Entities">CNPJ</abbr>s
and the chance of running out of available numbers,
on October 15, 2024, the Brazilian Federal Revenue Service published a normative instruction
([Normative Instruction RFB No. 2229](https://normasinternet2.receita.fazenda.gov.br/#/consulta/externa/141102))
changing the format of the <abbr title="National Registry of Legal Entities">CNPJ</abbr>.
The new format will now include letters and numbers.

The new number will continue to have 14 positions. The first twelve will be made up of
letters and numbers and the last two corresponding to the verification digits will remain numeric.

## Compatibility

The new format is compatible with the previous one and all existing
numeric <abbr title="National Registry of Legal Entities">CNPJ</abbr>s will remain
valid.

## When does it come into force?

According to the current Normative Instruction, the forecast is July 2026, if there are no postponements.

## Algorithm

### Calculating the check digits

The alphanumeric <abbr title="National Registry of Legal Entities">CNPJ</abbr> is made up
of twelve alphanumeric characters and two numeric check digits. The check digits (CD)
are calculated from the first twelve characters in two steps.

## Technical adjustments

### Database

The <abbr title="National Registry of Legal Entities">CNPJ</abbr> fields need to be
set to alphanumeric if they are numeric.

### Interface

#### Validation mask

As mentioned, the mask of the new <abbr title="National Registry of Legal Entities">CNPJ</abbr>
has been changed, so it will be necessary to update the validation mask of the `<input>`.

The new pattern:

`[A-Z0-9]{2}.[A-Z0-9]{3}.[A-Z0-9]{3}/[A-Z0-9]{4}-[0-9]{2}`

#### Validation algorithm

The validation algorithm needs to be updated to validate an alphanumeric <abbr title="National Registry of Legal Entities">CNPJ</abbr>.

To make things easier, I've released a TypeScript lib with the new algorithm
([gerador-validador-cnpj](https://github.com/tiagoporto/gerador-validador-cpf/tree/main/packages/gerador-validador-cnpj)).

![](/blog/cnpj/generator-validador-cnpj.png)

Install with <abbr title="Node Package Manager">NPM</abbr>:

```bash
npm install gerador-validador-cnpj -D
```

Use:

```js
import { validate as validadeCNPJ } from 'gerador-validador-cnpj'

const isCNPJValid = validadeCNPJ('01234567890000')
const isCNPJAlphanumericValid = validadeCNPJ('9ZW2JIM2OWTG85', {
  validateAlphanumeric: true
})
const isFormattedCNPJValid = validadeCNPJ('01.234.567/8900-00')
const isFormattedCNPJAlphanumericValid = validadeCNPJ('8I.S4O.LPO/PRD7-81', {
  validateAlphanumeric: true
})
```

If you're using Deno, you don't need to install it, just import:

```ts
import { validate } from 'jsr:@tiagoporto/gerador-validador-cnpj'
```

Complete documentation [<abbr title="Node Package Manager">NPM</abbr>](https://www.npmjs.com/package/gerador-validador-cnpj)
or [<abbr title="JavaScript registry" >JSR</abbr>](https://jsr.io/@tiagoporto/gerador-validador-cnpj).
